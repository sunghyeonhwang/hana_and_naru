# 하나와 나루의 세계사 모험

중학교 2학년 세계사 학습을 위한 **청소년 판타지 소설 시리즈**. 곰인형 나루의 시간 회중시계를 타고 하나가 세계사 속 주요 시대를 여행합니다.

> 창비 청소년문학의 단단한 문장력과 신카이 마코토 『너의 이름은』식 따뜻한 판타지를 결합한 하이브리드 스타일.

## 시리즈 구성

| 편 | 제목 | 범위 | 페이지(A5) |
| --- | --- | --- | --- |
| 1편 | 강물의 첫 글자, 별의 마지막 문장 | 선사 문화 ~ 고대 제국 ~ 크리스트교 탄생 | 46p |
| 2편 | 네 개의 빛이 번지는 밤 | 불교·힌두교 / 동아시아 / 이슬람 / 중세 크리스트교 ~ 종교개혁 | 42p |

## 파일 안내

```
소설/
├── 1편-고대문명/
│   ├── story.md            # 원본 마크다운 (HTML 스팬으로 색 표기)
│   ├── 1편-고대문명.pdf       # A5 판형 PDF (표지 + 본문)
│   └── 하나와 나루 1권.epub   # EPUB 3 (표지 포함)
├── 2편-문화권의확산/
│   ├── story.md
│   ├── 2편-문화권의확산.pdf
│   └── 하나와 나루 2권.epub
├── md2pdf.py               # Chrome 헤드리스로 MD → PDF 변환
└── md2epub.py              # stdlib만으로 MD → EPUB 3 빌드
```

## 색상 규칙

학습 효과를 위해 본문에 세 단계의 강조가 들어 있습니다.

| 색 | 의미 | 표기 |
| --- | --- | --- |
| 검정 | 일반 서술 | `일반 문장` |
| <span style="color:#1e6fd9">**파랑**</span> | 핵심 용어 (교과서의 파란색 용어) | `<span style="color:#1e6fd9;font-weight:bold">용어</span>` |
| <span style="color:#dc2626">**★빨강**</span> | 최고 중요 용어 (교과서의 빨간색 ★ 용어) | `<span style="color:#dc2626;font-weight:bold">★용어</span>` |

원본 학습 자료의 색깔을 그대로 유지해서, 학생이 이야기를 읽는 동안 자연스럽게 시험 출제 포인트를 시각적으로 기억하도록 설계되었습니다.

## PDF 생성

```bash
python3 소설/md2pdf.py \
  소설/1편-고대문명/story.md \
  소설/1편-고대문명/1편-고대문명.pdf \
  --cover "하나와 나루 표지 1편.jpeg"
```

- A5 판형 (148×210mm), 본문 11pt, 줄간격 1.85
- Apple SD Gothic Neo 우선, 한글 keep-all 줄바꿈
- Chrome 헤드리스 렌더링 (색/폰트 100% 일관)

## EPUB 생성

```bash
python3 소설/md2epub.py \
  소설/1편-고대문명/story.md \
  소설/1편-고대문명/하나와\ 나루\ 1권.epub \
  --cover "하나와 나루 표지 1편.jpeg" \
  --title "1편 — 강물의 첫 글자, 별의 마지막 문장" \
  --author "하나와 나루의 세계사 모험"
```

- EPUB 3 (EPUB 2 레거시 표지 메타 동시 기재)
- 외부 의존성 없이 Python stdlib만 사용
- Apple Books, Calibre, ReadEra 등에서 호환 확인

## 캐릭터 바이블

- **하나** — 중학교 2학년 여학생. 역사 시험 앞에서는 한숨부터 나오는 평범한 학생. 감정을 크게 드러내지 않는 1인칭 현재시제 화자.
- **나루** — 하나의 갈색 곰인형. 하나 앞에서만 움직이고 말함. 배에 달린 구릿빛 회중시계가 시간 회중시계. 장난스럽고 철없는 동생 같은 존재이면서도 이상하게 역사 지식은 해박함.
- **시간 회중시계** — 나루가 가고 싶은 시대를 속삭이고 용두를 돌리면, 별 가루 같은 빛이 두 사람을 감싸며 시공간을 건너뜀.
